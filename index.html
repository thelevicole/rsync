Ignore files

Define patterns to exclude files. One pattern per line. For example, enter *.png in the upper part to exclude all PNG files, and enter /assets/logo.png in the lower part to make an exception for the logo file.

Some examples...

config All files and directories named 'config' together with their contents
*.png All PNG files
/path/name File or directory together with its content in a specific path
name* All files starting with name and all directories starting with name together with their contents
